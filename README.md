# Clerk

A desktop assistant that keeps receipts.

Clerk is a governed desktop agent for non-developers. It manages files,
runs tasks, and chats — and every action it takes is evidence-gated,
receipted, and auditable.

Built on [Agent Governor](https://github.com/unpingable/agent_governor)
(14,000+ tests, 60+ governance modules, receipt kernel).
